# v0.3 Release Notes

## 定位

v0.3 是在 v0.2 可打印资料包基础上的“签到题与常识兜底”增强版，重点补齐容易在机考中被包装成模拟题、AI 背景题或基础常识题的内容。

## 主要新增

- 新增第 11 卷：签到题百科，单卷 `58` 页。
- 新增 Markov 性质专题：Markov 链、转移矩阵、矩阵快速幂、平稳分布、吸收状态、HMM/MDP 路由。
- 新增现代 AI 术语专题：Token、Embedding、Attention、Transformer、RAG、向量检索、训练/验证/测试、Top-K 检索。
- 新增计算机数值与内存表示：进制、补码、浮点、字节序、内存区、结构体对齐、BCD、固定小数。
- 新增 NOIP/CSP 初赛式读程序专题：流程图、变量表、循环次数、递归栈、短路求值、函数传参。
- 扩充常用数学、微积分、线性代数、概率统计、机器学习指标、媒体文件大小、OS/网络/数据库、安全和信息论常识。
- 第 6 卷补充 SIM 方程求解专题：高斯消元、二分求根、牛顿法和相关考场策略。
